## 🎯 个性化定制

### 修改表白文字
打开 `index.html`，找到：
```html
<h1 class="title">遇见你，是我最美的意外</h1>
<p class="subtitle">从遇见你的那一刻起，我的世界就变得不一样了</p>
```

### 修改颜色主题
在 `styles.css` 中修改：
```css
/* 背景色 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 爱心颜色 */
background: #ff0066;
```
